# Automação Web com Cypress

Projeto desenvolvido para praticar automação de testes web utilizando Cypress e JavaScript.

## Objetivo

Automatizar cenários de login e cadastro, aplicando boas práticas de automação de testes e organização do código.

## Tecnologias utilizadas

- Cypress
- JavaScript
- Node.js
- Git
- GitHub

## Funcionalidades implementadas

### Login

- Login com credenciais válidas
- Login com senha inválida
- Login com e-mail inválido
- Login com credenciais vazias

### Cadastro

- Cadastro realizado com sucesso
- Cadastro com e-mail inválido
- Cadastro com senha inválida
- Cadastro com nome vazio
- Cadastro com e-mail vazio
- Cadastro com senha vazia

## Melhorias implementadas

- Configuração da `baseUrl` no Cypress.
- Utilização do `beforeEach()` para reduzir repetição de código.
- Organização dos testes em arquivos separados por funcionalidade.
- Correção do cenário de login com credenciais vazias.

## Próximas implementações

- Utilização do FakerJS para geração de dados dinâmicos.
- Implementação de Page Object Model (POM).
- Custom Commands.
- Fixtures.
- Relatórios de execução.
- Evidências dos testes.
